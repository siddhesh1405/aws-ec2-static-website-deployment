## Overview
This project demonstrates the deployment of a static website on an AWS EC2 instance using Nginx.  
The project follows best practices including SSH-based GitHub authentication and version control.

## Architecture
- AWS EC2 (Amazon Linux)
- Nginx Web Server
- Git & GitHub (SSH authentication)
- HTML

## Key Features
- EC2 instance provisioning with security group configuration
- Nginx web server installation and configuration
- SSH key-based GitHub authentication from EC2
- Version-controlled deployment workflow

## Steps Performed
1. Launched EC2 instance and configured security groups
2. Installed and configured Nginx web server
3. Generated SSH keys and connected GitHub to EC2
4. Created and deployed a static website
5. Managed code using Git and GitHub
